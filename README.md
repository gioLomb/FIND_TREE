<b>FIND TREE</b>
<BR>
Questo piccolo progetto nasce per necessità personale : orientarsi nei terreni di campagna non è facile e avevo bisogno di un aiuto per orientarmi al meglio.<br>
Per facilitare il suo utilizzo usufruisco di NGROK.
<br>Dopo aver avviato il sito, sarà visibile la propria posizione, oltre a quella di tutti gli alberi registrati.<br>
E' possibile anche avviare una sorta di indicazione dalla propria posizione a un albero selezionato.<br>
Infine è possibile segnalare un pericolo con un semplice tasto che farà suonare un allarme
